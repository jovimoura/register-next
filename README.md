<br>

## 🇺🇸 ENG

### Clone😎

```
git clone https://github.com/jovimoura/register-next
```

### Access the folder🤓

```
cd <folder>
```
### Install dependencies🤠
```
npm install
```
### Start application🤩
```
npm run serve
```
### default port:🤗

```
http://localhost:3000/
```

<br>
